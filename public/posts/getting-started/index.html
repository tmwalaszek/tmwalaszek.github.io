Getting Started with Hugo

January 15, 2024 hugo blogging tutorial

Welcome to your new Hugo blog! This is your first post written in Markdown.

What is Hugo?

Hugo is a fast and modern static site generator written in Go. It’s perfect for building blogs, portfolios, and documentation sites.

Key Features

  • Blazing fast build times
  • Markdown support out of the box
  • Simple and clean structure
  • No database required

Writing in Markdown

Markdown makes it super easy to format your content. Here are some basics:

Text Formatting

You can use italic, bold, or bold italic text. You can also use strikethrough.
